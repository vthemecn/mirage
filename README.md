## ✨ 功能特性

### 🎨 设计与体验
- **现代化设计** - 简洁优雅的 UI 设计，注重用户体验
- **响应式布局** - 完美适配桌面、平板、手机等各种设备
- **暗黑模式** - 支持自动检测系统主题，提供舒适的夜间浏览体验
- **自定义配色** - 可自定义主题主色调，满足个性化需求

### 🌐 国际化支持
- **多语言支持** - 内置中文、英文、日文翻译
- **易于扩展** - 可轻松添加更多语言支持

### ⚡ 性能优化
- **无刷新加载** - 首页内容无刷新加载，提升用户体验
- **原生开发** - 使用原生 CSS 和 ES6+ 开发，无前端框架依赖，加载更快
- **优化的代码结构** - 基于 WordPress 最佳实践构建

### 🔐 用户系统
- **用户中心** - 完整的用户个人中心功能
- **文章管理** - 用户可在前端发布和管理自己的文章
- **权限控制** - 支持内容可见性控制（登录可见、评论可见、VIP可见）
- **防暴力破解** - 后台防暴力登录功能

### 📝 内容功能
- **文章互动** - 支持点赞、收藏、海报分享
- **多媒体支持** - 丰富的媒体管理和展示功能
- **SEO 优化** - 内置 SEO 功能，提升搜索引擎友好性
- **内容目录** - 自动生成文章内容目录（TOC）

### 🛠️ 主题设置
- **可视化设置面板** - 基于 Codestar Framework 的强大设置面板
- **导入导出配置** - 可备份和恢复主题设置
- **SMTP 设置** - 自定义 SMTP 配置
- **多种小工具** - 丰富的预设小工具

## 安装
在发行版页面下载 .zip 文件，在 WordPress 后台 "主题" 页面上传并安装。

提示：请不要直接克隆或下载仓库进行使用，请到发行版中下载使用。
