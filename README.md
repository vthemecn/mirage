## ✨ 功能特性

### 🎨 设计与体验
- **现代化设计** - 简洁优雅的 UI 设计，注重用户体验
- **响应式布局** - 完美适配桌面、平板、手机等各种设备
- **暗黑模式** - 支持自动检测系统主题，提供舒适的夜间浏览体验
- **自定义配色** - 可自定义主题主色调，满足个性化需求

### 🌐 国际化支持
- **多语言支持** - 内置中文、英文、日文翻译
- **易于扩展** - 可轻松添加更多语言支持

### ⚡ 性能优化
- **无刷新加载** - 首页内容无刷新加载，提升用户体验
- **原生开发** - 使用原生 CSS 和 ES6+ 开发，无前端框架依赖，加载更快
- **优化的代码结构** - 基于 WordPress 最佳实践构建

### 🔐 用户系统
- **用户中心** - 完整的用户个人中心功能
- **文章管理** - 用户可在前端发布和管理自己的文章
- **权限控制** - 支持内容可见性控制（登录可见、评论可见、VIP可见）
- **防暴力破解** - 后台防暴力登录功能

### 📝 内容功能
- **文章互动** - 支持点赞、收藏、海报分享
- **多媒体支持** - 丰富的媒体管理和展示功能
- **SEO 优化** - 内置 SEO 功能，提升搜索引擎友好性
- **内容目录** - 自动生成文章内容目录（TOC）

### 🛠️ 主题设置
- **可视化设置面板** - 基于 Codestar Framework 的强大设置面板
- **导入导出配置** - 可备份和恢复主题设置
- **SMTP 设置** - 自定义 SMTP 配置
- **多种小工具** - 丰富的预设小工具

## 🚀 快速开始

### 环境要求
- WordPress 6.1+
- PHP 7.3+
- MySQL 5.6+ 或 MariaDB 10.0+

### 安装

#### 方法一：通过 WordPress 后台
1. 访问 WordPress 后台 > 外观 > 主题
2. 点击"添加新主题" > "上传主题"
3. 选择下载的 Mirage 主题压缩包
4. 点击"立即安装"

#### 方法二：FTP 上传
1. 解压主题压缩包
2. 将整个 `mirage` 文件夹上传到 `/wp-content/themes/` 目录
3. 在 WordPress 后台激活主题

### 配置
1. 激活主题后进入"主题设置"页面
2. 根据需要调整各项配置
3. 保存设置并预览效果
